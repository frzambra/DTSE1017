1.1 Iteración for vs funciones vectorizadas

  1. Cree una función que mediante un ciclo for haga la suma de los n elementos de un vector. Aplique la función sobre el vector x <- rnorm(1e9). Use la función system.time para conocer el tiempo quese demoro en hacer la operación.

  2. Utilice la operación vectorizada sum para hacer la suma del vector x. Use la función system.time para conocer el tiempo quese demoro en hacer la operación.

  3. Cuál de las dos alternativas realiza el cálculo en menor tiempo.
